# Doubao Account Manager

一个支持多平台的账号管理工具，**逆向工程友好型项目**。

## 🔓 核心理念

> **开源精神，逆向实现。**

本项目采用一种独特的方式来分享技术：
- ✅ **完全开放授权**：允许所有人下载、使用、反编译、逆向工程
- 🔍 **不提供源代码**：所有代码都需要通过逆向工程来发现
- 🎯 **彩蛋挑战**：软件中藏有多个彩蛋，等待被发现
- 📚 **学习交流**：提供一个逆向工程的实战练习平台

## ⚡ 功能特性

- 支持多平台账号管理
- 视频无水印下载
- 支持豆包、Dola、千问、即梦等平台
- 简洁易用的界面

## 📦 快速开始

1. 下载最新版本的 `AccountManager*.zip`
2. 解压到任意目录
3. 运行 `DoubaoAccountManager*.exe`
4. 开始使用（或开始逆向工程！）
