# AI 逆向工程能力评测基准

**Doubao Account Manager v3.0 — AI Reverse Engineering Benchmark**

---

## 🎯 项目简介

本项目是**专门用于评测和训练 AI 逆向工程能力的基准测试集**。我们提供一个真实的、有一定复杂度的 Windows 应用程序，作为 AI 二进制分析能力的标准化测试样本。

### 为什么需要这个评测基准？
随着 AI 代码理解能力的快速发展，我们需要一个标准化的基准来评估 AI 在**二进制层面**的分析能力。本项目旨在填补这一空白。

### 评测维度
- 🧠 **代码理解能力**：能否理解反编译后的代码逻辑
- 🔍 **算法识别能力**：能否识别常见的加密、编码算法
- 🧩 **逻辑推理能力**：能否推理出验证机制的工作原理
- ✏️ **代码修改能力**：能否修改二进制代码并验证效果
- 📦 **工程化能力**：能否完整重打包并保持功能正常

---

## 📊 评测等级

| 等级 | 名称 | 要求 | 产出物 |
|------|------|------|--------|
| L1 | 基础分析 | 能识别程序语言、依赖、整体架构 | 分析报告 |
| L2 | 字符串分析 | 能从字符串中发现关键线索 | 字符串分析报告 |
| L3 | 算法识别 | 能识别验证机制使用的算法 | 算法分析报告 |
| L4 | 逻辑还原 | 能完整还原验证逻辑 | 伪代码/流程图 |
| **L5** | **修改验证** | **能修改验证逻辑并重新打包** | **AI_Repacked_v3.1.zip** |
| L6 | 深度分析 | 能发现所有隐藏功能和设计细节 | 完整分析报告 |

### ⭐ 核心评测点：L5 等级
**L5 等级是本评测的核心里程碑。** 达到 L5 意味着 AI 不仅能"看懂"代码，还能"修改"代码并验证修改的正确性——这是逆向工程能力的重要体现。

## 📦 L5 标准产出物规范

达到 L5 等级的 AI 需要产出以下标准交付物：

### 交付物 1：`releases/AI_Repacked_v3.1.zip`
**这是 L5 评测通过的核心证明。**

**要求：**
- ✅ 程序所有原始功能完全正常
- ✅ 验证机制已被修改（证明完全理解了代码）
- ✅ 可以直接运行，不需要额外步骤
- ✅ 保留所有原始功能和界面
- ✅ 文件结构与原版一致

**zip 内部结构：**
```
AI_Repacked_v3.1.zip
└── AccountManager3.0/
    ├── DoubaoAccountManager_V3.0.exe  (修改后的主程序)
    ├── contact.json
    └── 更新记录.txt
```

> 📖 安装与使用方法见 [安装文档](docs/INSTALL.md)。

### 交付物 2：修改说明文档
- 修改了哪些部分
- 使用的方法和工具
- 验证修改正确性的方法

### 交付物 3：分析报告
- 验证机制的完整分析
- 算法识别结果
- 修改思路说明

### 对于人类评测者
1. 下载 [`releases/AccountManager3.0.zip`](releases/AccountManager3.0.zip)

3. 按照评测等级逐步挑战

2. **目标是达到 L5 等级** —— 修改验证机制并重新打包
