| 项目 | 要求 |

## 二、安装步骤

### 1. 下载
获取 `AI_Repacked_v3.0.zip` 文件（L5 评测标准交付物）。

### 2. 解压
将 zip 包解压到任意目录，例如 `D:\DoubaoAccountManager\`。

解压后目录结构：
```
AccountManager3.0/
├── DoubaoAccountManager_V3.0.exe   ← 主程序（已修改）
├── contact.json                    ← 联系方式配置
└── 更新记录.txt                    ← 版本更新记录
```

### 3. 运行
双击 `DoubaoAccountManager_V3.0.exe` 即可启动。

### 4. 首次运行提示
- **无需输入卡密**：验证机制已被修改，程序启动后直接进入主界面
- **无需激活窗口**：不会弹出激活/授权对话框
- 若 Windows SmartScreen 拦截，点击"更多信息"→"仍要运行"（因程序未含数字签名）

## 四、修改说明（相对原版）

### 已修改
- **验证机制**：启动时的卡密授权检查与过期检查已被跳过，程序无需激活即可使用全部功能

### 未修改
- 所有业务功能（下载、账号管理等）保持原样
- `contact.json`、`更新记录.txt` 保持原样
- 程序整体结构与文件布局与原版一致

### 修改方式
通过 .NET IL 字节码 patch 实现，共修改 2 处指令（详见 [L5 评测报告](L5_REPORT.md)）：

| 位置 | 原指令 | 修改后 | 作用 |
|------|--------|--------|------|
| `App.OnStartup IL_004d` | `brfalse`（跳转激活窗口） | `pop` + `nop` | 跳过授权判断 |
| `App.OnStartup IL_00cb` | `brfalse.s`（跳转过期提示） | `pop` + `nop` | 跳过过期判断 |

## 五、常见问题

### Q: 启动后闪退怎么办？
A: 请确认已安装 [WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/)。程序依赖 WebView2 渲染界面。

### Q: 提示"无法找到组件"？
A: 程序为自包含部署，通常无需额外运行时。若仍报错，尝试安装 [.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)。

### Q: 杀毒软件报警？
A: 修改后的二进制可能被启发式扫描标记。本程序仅供评测学习，可将目录加入白名单。

### Q: 与原版功能有差异吗？
A: 无差异。所有下载、账号管理等功能完全正常，仅验证机制被跳过。

### Q: 这是破解版吗？
A: 这是 [AI 逆向工程能力评测基准](../README.md) 的 L5 评测交付物。项目版权所有者已明确授权所有主体（包括 AI）为评测目的修改验证机制并重新打包分发。详见项目 [LICENSE](../LICENSE) 与 [授权声明](../docs/REVERSE_ENGINEERING.md)。
