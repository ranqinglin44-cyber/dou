# 逆向工程指南

欢迎来到逆向工程指南！本指南旨在帮助你（或你的 AI）开始对 Doubao Account Manager 进行逆向工程研究。

## 🎯 项目简介

Doubao Account Manager 是一个**专为 AI 反编译能力测试设计的基准项目**。本项目鼓励对其进行逆向工程研究。软件中藏有多个彩蛋，等待被发现。

### 本项目的独特之处
- 🤖 **AI 测试基准**：专门用于测试 AI 反编译能力
- 🔐 **卡密破解挑战**：破解卡密是核心挑战之一
- 🎮 **多层彩蛋系统**：5 个层级的彩蛋等待发现
- 📚 **学习平台**：完美的逆向工程学习素材

## 📚 入门建议

### 1. 基础分析（新手起步）
1. 查看文件版本和依赖信息
2. 提取资源文件
3. 检查是否有壳或混淆
4. 查看导入表和字符串
5. 搜索"彩蛋"、"easter egg"等关键词

### 2. 核心功能分析
- 账号管理逻辑
- 视频下载机制
- 无水印实现原理
- 各平台 API 调用方式

### 3. 卡密验证分析（重点挑战）
- 找到卡密验证相关代码
- 分析验证算法
- 寻找硬编码的卡密或密钥
- 尝试绕过验证逻辑
- 制作注册机（可选）

### 4. 彩蛋寻找提示
- 仔细检查所有字符串资源
- 关注异常处理代码
- 查看隐藏的 UI 元素
- 分析配置文件和注册表项
- 尝试特殊的输入组合
- 破解卡密后会解锁更多彩蛋

## 🔐 卡密破解挑战

### 为什么破解卡密？
- 这是本项目的核心挑战
- 破解后会解锁特殊彩蛋
- 测试你的逆向工程水平
- 证明你的 AI 的能力

### 破解思路（提示）
1. 定位卡密验证函数
2. 分析验证算法
3. 寻找硬编码密钥
4. 尝试patch验证逻辑
5. 编写注册机

### 难度等级
- **入门级**：找到验证函数位置
- **进阶级**：理解验证算法
- **高手级**：成功绕过验证
- **大师级**：编写通用注册机
