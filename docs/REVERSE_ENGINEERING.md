# 逆向工程指南

欢迎来到逆向工程指南！本指南旨在帮助你开始对 Doubao Account Manager 进行逆向工程研究。

## 🎯 项目简介

Doubao Account Manager 是一个账号管理工具，本项目鼓励对其进行逆向工程研究。软件中藏有多个彩蛋，等待被发现。

## 🔓 授权声明

**本项目明确授权所有人进行逆向工程研究，包括但不限于：**
- ✅ 反编译
- ✅ 调试分析
- ✅ 内存分析
- ✅ 网络抓包
- ✅ 算法还原
- ✅ 二次开发

## 🛠️ 推荐工具

### 静态分析
- **dnSpy / ILSpy** - .NET 反编译工具
- **Ghidra** - 逆向工程套件
- **IDA Pro** - 专业反汇编工具
- **Resource Hacker** - 资源查看编辑工具

### 动态分析
- **x64dbg / OllyDbg** - 调试器
- **Wireshark** - 网络抓包
- **Fiddler / Charles** - HTTP/HTTPS 抓包
- **Process Monitor** - 系统调用监控

### .NET 专用
- **dnSpy** - 调试 + 反编译
- **de4dot** - 脱壳/混淆工具
- **ClrMD** - .NET 内存分析

## 📚 入门建议

### 1. 基础分析
1. 查看文件版本和依赖信息
2. 提取资源文件
3. 检查是否有壳或混淆
4. 查看导入表和字符串

### 2. 核心功能分析
- 账号管理逻辑
- 视频下载机制
- 无水印实现原理
- 各平台 API 调用方式

### 3. 彩蛋寻找提示
- 仔细检查所有字符串资源
- 关注异常处理代码
- 查看隐藏的 UI 元素
- 分析配置文件和注册表项
- 尝试特殊的输入组合
