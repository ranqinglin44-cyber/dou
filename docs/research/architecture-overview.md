## 模块划分

根据初步分析，软件大致分为以下模块：

### 1. 账号管理模块
- 多平台账号存储
- 账号加密存储
- 账号切换功能

### 2. 视频下载模块
- 多平台支持（豆包、Dola、千问、即梦）
- 无水印处理
- 批量下载功能

### 3. UI 界面模块
- 主窗口
- 设置页面
- 关于对话框

### 4. 网络通信模块
- API 请求封装
- 认证处理
- 错误重试机制

## 数据流

```
用户输入 → UI层 → 业务逻辑层 → 网络层 → API接口
                ↓
           数据持久化层
                ↓
         本地配置/缓存文件
```
