# 推荐工具配置指南

本文档介绍逆向工程常用工具的推荐配置，帮助你快速开始。

## 🛠️ .NET 反编译工具

### dnSpy
- **用途**: .NET 反编译 + 调试
- **推荐版本**: 最新版
- **配置建议**:
  - 启用"显示所有成员"
  - 启用"反编译优化"
  - 配置符号服务器
- **快捷键**:
  - `F5` - 开始调试
  - `F9` - 切换断点
  - `F12` - 转到定义

### ILSpy
- **用途**: 轻量级 .NET 反编译
- **推荐版本**: 最新版
- **插件推荐**:
  - ILSpy.BamlDecompiler
  - ILSpy.ReadyToRun

## 🔍 通用逆向工具

### Ghidra
- **用途**: 通用逆向工程套件
- **推荐版本**: 最新版
- **配置建议**:
  - 安装 .NET 解析插件
  - 配置 Python 脚本目录

### x64dbg
- **用途**: 动态调试
- **推荐版本**: 最新版
- **推荐插件**:
  - ScyllaHide
  - x64dbg Scripts Collection

## 🌐 网络分析工具

### Fiddler
- **用途**: HTTP/HTTPS 抓包
- **配置建议**:
  - 启用 HTTPS 解密
  - 配置过滤器
  - 安装自定义脚本

### Wireshark
- **用途**: 网络协议分析
- **过滤器技巧**:
  - `http` - 只看 HTTP 流量
  - `tcp.port == 443` - 只看 HTTPS

## 📦 资源编辑工具

### Resource Hacker
- **用途**: 查看/编辑 Windows 资源
- **常用功能**:
  - 查看对话框资源
  - 提取图标
  - 修改字符串

## 📝 其他实用工具

### Process Monitor
- **用途**: 监控系统调用
- **过滤器建议**:
  - 按进程名过滤
  - 关注文件和注册表操作

### Dependencies
- **用途**: 查看 DLL 依赖
- **替代方案**: Dependency Walker
