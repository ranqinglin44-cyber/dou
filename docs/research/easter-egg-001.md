## 描述

在软件的字符串资源中隐藏着一条开发者留言，需要通过反编译才能看到。

## 发现过程

### 方法一：字符串搜索
1. 使用 dnSpy 或 ILSpy 打开程序集
2. 搜索字符串资源
3. 查找异常的、看起来不像正常功能的字符串
4. 你会找到一条有趣的开发者留言

### 方法二：资源文件提取
1. 使用 Resource Hacker 打开 exe
2. 查看字符串资源表
3. 仔细浏览所有字符串
4. 发现隐藏的留言
